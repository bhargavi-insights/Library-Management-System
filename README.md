## 🚀 Overview

> A data-driven system to manage books, members, and borrowing operations in a structured and efficient way.

✔ Supports book tracking  
✔ Manages members  
✔ Handles borrow & return flow  
✔ Stores data using JSON  

---

## ✨ Features

- 📖 Add new books  
- 📋 View all books  
- 👤 Add members  
- 👥 View members  
- 🔄 Borrow books  
- ✅ Return books  
- 💾 Persistent storage
